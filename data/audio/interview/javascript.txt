CANDIDATE: Undefined means a variable was declared but never assigned, it's the engine's default, while null is a value you deliberately set to say there is intentionally no value here. CANDIDATE: A closure is a function that remembers the variables from the scope where it was created, even after that scope has returned. The why is that this is how JavaScript gives you private state without classes, the inner function keeps a live link to its outer variables.
